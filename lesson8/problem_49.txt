
Телефонный справочник.
Реализовано:
 1. Просмотр всего справочника.
 2. Добавление записи.
 3. Сохранение изменений в файл.
 3. Поиск записи. Находит первую запись, отвечающего условию. Из поиска доступны:
    3.1 Изменение найденной записи. Поля, которые не нужно изменять, можно пропускать без изменения, нажимая ENTER.
    3.2 Удаление найденной записи.
    После изменения записи или удаления записи изменения без запроса сохраняются в файл.
    После удаления записи 1е поле (нумерация) перенумеровывается.
    Если 1 поле - ID, а не порядковый номер, - функцию перенумерации следует отключить.

Не реализовано:
 1. Наименование полей, использование имён полей для облегчение ввода/редактирования.
 2. Поиск нескольких записей с одинаковыми данными, дальнейший выбор из найденых записей для редактирования и удаления.
 3. Поиск по неполным данным.
